# Cellular automatons visualizer

This python software allows to visualize one-dimensional cellular automatons graphically.

User input:
1. CA Rule (8-bit representation of Wolfram code eg. 00011101)
2. Length of configuration N (eg. 15)
3. Starting configuration x_0 (eg. 000110100100111)
4. Number of iterations (eg. 100)

## Run:

In source directory type in terminal:
`uv run python3 main.py`
